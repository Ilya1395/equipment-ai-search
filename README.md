# Поиск характеристик моделей оборудования

Streamlit-приложение для поиска по открытым источникам характеристик оборудования и формирования таблицы.

## Что делает приложение

Пользователь вводит:

- `Класс`, например `Насосы`;
- `Подкласс`, например `Погружные`;
- `Код модели`, например `ГНОМ 40-25`.

Далее приложение:

1. ищет в выбранном поисковике сочетание `Класс + Подкласс + Код модели`;
2. поочередно открывает найденные сайты;
3. ищет на странице код модели;
4. извлекает характеристики, значения и единицы измерения;
5. заполняет итоговую таблицу;
6. добавляет в седьмой столбец `Источник` название сайта с гиперссылкой;
7. позволяет скачать результат в CSV и XLSX.

## Поисковики

В боковой панели можно выбрать:

- Google.com;
- Ya.ru;
- Yahoo.com.

Между запросами и открытием страниц установлена задержка 2 секунды.

## Нейросети

В приложении оставлены бесплатные open-source модели Hugging Face:

- Qwen 2.5 7B Instruct;
- Mistral 7B Instruct v0.3;
- Zephyr 7B Beta;
- Falcon 7B Instruct;
- Phi-3 Mini 4K Instruct;
- Gemma 2 2B IT.

Фактическая доступность зависит от лимитов Hugging Face и прав доступа к конкретной модели.

Если `HF_TOKEN` не задан, приложение использует резервное извлечение характеристик по регулярным выражениям.

## Секреты Streamlit

В Streamlit Cloud откройте настройки приложения и добавьте в Secrets:

```toml
HF_TOKEN = "hf_ВАШ_ТОКЕН"
```

## Файлы проекта

```text
app.py                  # основной файл Streamlit-приложения
requirements.txt        # зависимости Python
runtime.txt             # рекомендуемая версия Python
utils/search.py         # поиск через Google, Ya.ru, Yahoo и загрузка страниц
utils/llm.py            # извлечение характеристик через Hugging Face или резервный алгоритм
utils/normalize.py      # нормализация единиц измерения и удаление дублей
.streamlit/secrets.toml.example  # пример секретов
```

## Запуск в Streamlit Cloud

1. Загрузите файлы проекта в GitHub.
2. В Streamlit Community Cloud создайте новое приложение из этого репозитория.
3. В поле `Main file path` укажите:

```text
app.py
```

4. В Secrets добавьте `HF_TOKEN`.
5. Нажмите `Deploy`.

## Важные ограничения

- Поисковые системы могут временно блокировать облачный сервер или показывать капчу.
- PDF-файлы напрямую не читаются в этой версии, но их заголовок и сниппет из поисковой выдачи используются.
- Бесплатные модели Hugging Face могут отвечать медленно или быть временно недоступны.
- При анализе 50 источников работа может занимать заметное время из-за задержки 2 секунды и последовательного открытия сайтов.


## Примечание к поиску

Если сайт или поисковик не отдает полный HTML, приложение все равно использует заголовок и сниппет найденного результата. Это снижает вероятность сообщения «Не удалось найти доступные источники».
