# Поиск характеристик моделей оборудования

Streamlit-приложение для поиска характеристик модели оборудования по открытым источникам в интернете и формирования таблицы.

Пользователь вводит:

- `Поле №1`, например `Насосы`;
- `Поле №2`, например `Погружные`;
- `Код`, например `ГНОМ 40-25`.

Приложение ищет сведения в открытых источниках, передает найденный текст выбранной бесплатной open-source нейросети через Hugging Face, извлекает характеристики и показывает таблицу:

1. Поле №1
2. Поле №2
3. Код
4. Характеристика
5. Значение
6. Ед. изм.

Результат можно скачать в `CSV` и `XLSX`.

## Что добавлено в этой версии

- Кнопка `Скачать XLSX`.
- Выбор нейросети в боковой панели.
- Возможность указать свою модель Hugging Face вручную.
- Обновленная инструкция для GitHub и Streamlit.

## Бесплатные компоненты

- Хостинг: Streamlit Community Cloud.
- Репозиторий: GitHub.
- Поиск: бесплатная Python-библиотека `ddgs`.
- Нейросеть: бесплатные open-source модели через Hugging Face Inference API / Inference Providers.
- XLSX: библиотека `openpyxl`.

Важно: бесплатные лимиты Hugging Face ограничены. Для демонстрации и небольшого личного использования этого обычно достаточно, но для промышленной нагрузки может понадобиться платный тариф или локальный сервер с моделью.

## Доступные нейросети в интерфейсе

В приложении есть выпадающий список:

- Qwen 2.5 7B Instruct: `Qwen/Qwen2.5-7B-Instruct`
- Mistral 7B Instruct v0.3: `mistralai/Mistral-7B-Instruct-v0.3`
- Zephyr 7B Beta: `HuggingFaceH4/zephyr-7b-beta`
- Falcon 7B Instruct: `tiiuae/falcon-7b-instruct`
- Phi-3 Mini 4K Instruct: `microsoft/Phi-3-mini-4k-instruct`
- Gemma 2 2B IT: `google/gemma-2-2b-it`

Если какая-то модель временно недоступна, выберите другую. Доступность зависит от Hugging Face, лимитов и условий конкретной модели. Некоторые модели могут потребовать принять лицензию на странице модели Hugging Face.

## Пошаговый запуск через GitHub и Streamlit

### Шаг 1. Создайте репозиторий GitHub

1. Зайдите в GitHub.
2. Нажмите **New repository**.
3. Назовите репозиторий, например `equipment-ai-search`.
4. Для простоты выберите **Public**.
5. Нажмите **Create repository**.

### Шаг 2. Загрузите файлы

1. Откройте созданный репозиторий.
2. Нажмите **Add file** -> **Upload files**.
3. Перетащите все файлы и папки из этого проекта.
4. Нажмите **Commit changes**.

Важно: загружать нужно содержимое папки проекта, а не сам ZIP-архив.

### Шаг 3. Получите бесплатный токен Hugging Face

1. Зарегистрируйтесь на Hugging Face.
2. Откройте настройки токенов: **Settings -> Access Tokens**.
3. Создайте токен с правом чтения.
4. Скопируйте токен, начинающийся с `hf_`.

### Шаг 4. Разверните приложение в Streamlit Community Cloud

1. Зайдите на Streamlit Community Cloud.
2. Войдите через GitHub.
3. Нажмите **New app**.
4. Выберите ваш репозиторий.
5. В поле **Main file path** укажите: `app.py`.
6. Откройте **Advanced settings** или настройки секретов.
7. Вставьте секреты:

```toml
HF_TOKEN = "hf_ВАШ_БЕСПЛАТНЫЙ_ТОКЕН"
```

8. Нажмите **Deploy**.

После запуска Streamlit выдаст отдельную публичную ссылку на приложение.

## Локальный запуск

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```

Для локального запуска создайте файл `.streamlit/secrets.toml`:

```toml
HF_TOKEN = "hf_ВАШ_БЕСПЛАТНЫЙ_ТОКЕН"
```

## Как пользоваться

1. Введите `Поле №1`, например `Насосы`.
2. Введите `Поле №2`, например `Погружные`.
3. Введите `Код`, например `ГНОМ 40-25`.
4. В боковой панели выберите нейросеть.
5. Нажмите `Найти характеристики`.
6. Проверьте таблицу.
7. Скачайте результат в `CSV` или `XLSX`.

## Ограничения бесплатной версии

- Некоторые сайты блокируют автоматическое чтение страниц.
- PDF-файлы в этой версии не распознаются напрямую.
- Бесплатный Hugging Face API может возвращать ошибки лимитов или временной недоступности модели.
- Результаты нужно проверять по источникам, потому что открытые данные по оборудованию могут отличаться у поставщиков и производителей.
- Нейросеть не должна выдумывать данные, но контроль результата все равно необходим.
