# CHANGELOG

## v7

- В итоговую таблицу добавлен седьмой столбец `Источник`.
- В столбце `Источник` отображается название сайта с кликабельной гиперссылкой.
- В CSV/XLSX источник сохраняется как `название сайта - URL`.
- Осветлены поля выбора и поля ввода: `Поисковик`, `Нейросеть для извлечения характеристик`, `Своя модель Hugging Face`, `Класс`, `Подкласс`, `Код модели`.
- Цвет пользовательского текста в этих полях изменен на белый.
- Алгоритм поиска изменен:
  1. формируется поисковый запрос `Класс + Подкласс + Код модели`;
  2. приложение поочередно открывает найденные сайты;
  3. для анализа оставляются страницы, где найден код модели;
  4. характеристики извлекаются отдельно по каждому источнику;
  5. итоговая таблица заполняется с привязкой характеристики к источнику.
