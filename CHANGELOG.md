# Changelog

## v6

- Добавлено поле выбора поисковика: `Google.com`, `Ya.ru`, `Yahoo.com`.
- Максимальное значение настройки `Количество источников для анализа` увеличено до 50.
- Поля формы переименованы:
  - `Поле №1` → `Класс`;
  - `Поле №2` → `Подкласс`;
  - `Код` → `Код модели`.
- Заголовок `Поиск характеристик моделей оборудования` уменьшен по размеру шрифта.
- Описание заменено на `Поиск по открытым источникам характеристик`.
- Добавлен CSS-интерфейс: голубой/васильковый фон, синие шрифты, светло-красные кнопки.
- Последние изменения с GigaChat, Z.ai и Alice не включены.

## v5

- Исправлен поиск через Ya.ru/Yandex.
- Добавлены резервные способы извлечения ссылок из поисковой выдачи.
- Убраны зависимости `ddgs`, `duckduckgo-search`, `lxml`.

## v4

- Добавлялись сторонние AI-провайдеры. В v6 они удалены по требованию.

## v3

- Поиск переведен на Ya.ru/Yandex с задержкой 2 секунды.

## v2

- Добавлена выгрузка XLSX.
- Добавлен выбор open-source модели Hugging Face.

## v1

- Базовое приложение Streamlit для поиска характеристик оборудования.
